# Changelog

All notable changes to FlyPlugin are documented in this file.

## [1.1.2] - 2026-07-11

### Changed

- Updated the Paper API dependency to `26.2.build.56-alpha`.
- Updated the declared plugin API version to `26.2`.
- Kept the Java compilation target at Java 25.

### Documentation - 2026-07-12

- Added the MIT License file.
- Added English SpigotMC resource-page and full-documentation BBCode.
- Added a manual SpigotMC publishing checklist.
- Documented the current command, permission, billing, configuration and data-storage limits.

## [1.1.1] - 2026-04-05

### Changed

- Updated the project for Paper 26.1 and Java 25.
- Corrected the Vault dependency declaration in `plugin.yml`.
- Enabled Maven resource filtering for the plugin version.
